## Getting Started

This is a backend of todoList app to install it locally do the following steps

### Prerequisites

Step 1: clone the repository

-git

```sh
git clone https://github.com/cabdiaziz/TodoApp-backend.git
```

Step 2: install the project

- npm
  ```sh
  npm install
  ```
- yarn

  ```sh
  yarn install
  ```

  Step 3: run

- npm
  ```sh
  npm start
  ```
