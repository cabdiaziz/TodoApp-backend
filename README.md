## Getting Started

This is a backend of todoList app to install it locally do the following steps

### Prerequisites

Clone the repository

- git

```sh
git clone https://github.com/cabdiaziz/TodoApp-backend.git
```

### Installation

Install the project

- npm

  ```sh
  npm install
  ```

- yarn

  ```sh
  yarn install / yarn
  ```

  Step 3: run

- npm

  ```sh
  npm start
  ```

- yarn
  ```sh
  yarn run
  ```
